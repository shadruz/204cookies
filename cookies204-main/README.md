# 204 Cookies

Веб-сайт для кондитерской "204 Cookies", предлагающей премиальное печенье с доставкой.

## Особенности

- Многоязычный интерфейс (Русский, O'zbekcha, English)
- Адаптивный дизайн
- Корзина покупок
- Каталог продукции
- Подарочные наборы
- Открытки
